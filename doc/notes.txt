Инструкция по прохождению всех уровней в Windows 7+

1. Чтобы пройти первый уровень (запустить переборщик паролей), нужно установить nodeJS вот отсюда:  http://nodejs.org/dist/v0.10.24/x64/node-v0.10.24-x64.msi. Так же, понадобиться библиотека exec-sync с сайта https://github.com/jeremyfa/node-exec-sync. Для запуска переборщика паaролей, нужно использовать такую запись в комадной строке: node BruteForce.js 7z.exe packedFilePath passwordLen. Файл bruteForce.js зависит от lib/js/Language.js.

2. Для прохождения второго уровня нужно взять, полученный из пункта 1 файл data.bin и скопировать его в папку JSQuest/2/js. После этого нужно запустить такую команду в консоли: node Interpreter.js .\..\scripts\unpacker.alan. Распаковка может занять до 10 минут (зависит от мощности машины).

3. Для прохождения 3 уровня (файл term.html) нужно запусть его в браузере Chrome и выполнить такую команду в консоли: connect jack "Love is".

4. Чтобы пройти 4 уровень нужно, найти землю (для этого нужно несколько раз ввести команду left 200). Когда она будет на экране (то есть в зоне видимости направленных передатчиков текущего спутника), нужно выполнить команду connect s1 s2 s3 s4 s5. После этого спутники должны загореться зеленым цветом. Во время работы земля должна находиться в зоне видимости, иначе произойдет отсоединение между текущим спутником и передатчиком на земле. Далее, нужно синхронизировать все файлы со всех спутников: sync s1 s2 s3 s4 s5. После выполнения этой команды, все файлы со всех спутников будут скопированы на текущий. Теперь нужно удалить эти файлы: remove .... Вместо трех точек нужно через пробел ввести все имен файлов, полученных с помощью команды list (cписок нужно получить после выполнения команды sync s1 s2 ...).

P.S. Для сборки проекта нужно запустить файл build\build.bat предварительно установив java и nodejs
